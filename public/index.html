🗺️ 下載流程一覽
🚀 快速開始（3 步下載第一條片）
  1. 1 登入系統（首次使用請向管理員取得帳號密碼）
  2. 2 點選 → 在右上方選擇分類（或保留「未分類」）→ 貼上 YouTube 網址 → 點擊 分析
  3. 3 選擇畫質（如 1080p）與格式（MP4 / MKV / MOV）→ 確認檔名日期前綴設定 → 點擊 開始下載
✅ 下載完成後會自動上傳至 Google Drive，並發送 Telegram 通知。無需手動操作。
🔐 登入 & 帳號管理
  1. 1 登入：輸入用戶名（電郵格式）及密碼，點擊 LOGIN。Session 有效期 24 小時，到期後需重新登入。
  2. 2 忘記密碼：點擊登入頁的 Forgot Password? → 輸入註冊電郵 → 系統發送 6 位數驗證碼 → 輸入驗證碼 → 顯示當前密碼（驗證碼 10 分鐘內有效，最多 5 次嘗試）
  3. 3 修改密碼： → 「帳號安全」→ 輸入目前密碼 + 新密碼 + 確認新密碼 → 儲存。儲存後所有登入 Session 自動清除，需重新登入。
  4. 4 登出： → 點擊「🚪 登出」按鈕
⚠️ 密碼儲存在伺服器環境變數，請勿將帳號資訊分享給他人。
📁 分類管理
  1. 1 新增分類：在輸入框輸入分類名稱（如「AI」、「音樂」）→ 點擊 + 新增。分類會建立對應的 Google Drive 子資料夾。
  2. 2 修改名稱：點擊分類卡片上的 ✏️ 圖示 → 直接在卡片上修改名稱 → 點擊 ✅ 確認。
  3. 3 刪除分類：點擊 🗑️ 圖示 → 確認刪除彈窗。注意：「未分類」為系統預設分類，無法刪除。
  4. 4 拖曳排序：按住左側 ⠿ 拖曳手柄，上下拖動即可重新排序分類。排序會即時同步至伺服器。
💡 分類會顯示在下載頁的下拉選單中，同時對應 Google Drive 的子資料夾結構。建議先建好分類再開始下載。
⬇️ 下載影片（完整流程）
  1. 1 選擇分類：在頁面左上角的下拉選單選擇目標分類（如「AI」）。此選擇決定影片儲存的 Google Drive 資料夾。
  2. 2 貼上 YouTube 網址：在輸入框貼上完整 YouTube 影片網址（支援 youtu.be 短網址及 youtube.com 長網址）。亦可直接按 Enter 觸發分析。
  3. 3 點擊「分析」：系統呼叫 yt-dlp 分析影片元數據（約 3-10 秒）。分析成功後顯示：影片縮圖、標題、頻道名稱、時長，以及各畫質選項與預計檔案大小。
  4. 4 選擇封裝格式：在「格式」區選擇 MP4 MKV 或 MOV。三者均為無損封裝（不重新編碼），檔案大小相同，建議使用 MP4（兼容性最佳）。
  5. 5 選擇畫質：在「畫質」區點選所需解析度（如 1080p）。右側顯示預計檔案大小（影片 + 音訊串流合計）。可用畫質取決於原片，最高可至 2160p（4K）。
  6. 6 日期前綴（選填）：若影片有上傳日期，頁面會顯示開關。啟用後檔名格式為 YYYYMMDD-標題.mp4（如 20260308-影片標題.mp4），方便按日期排序。預設開啟。
  7. 7 點擊「開始下載」：系統在背景啟動下載，頁面自動跳轉至「管理」Tab 追蹤進度。
💡 下載過程：yt-dlp 分別下載影片串流 + 音訊串流 → ffmpeg 合併 → 自動上傳 Google Drive → 發送 Telegram 通知。全程無需干預。
⚠️ 部分影片（年齡限制 / 地區限制）需要 YouTube Cookies 才能下載。請到 上傳 cookies.txt。
📊 下載管理
  1. 1 下載狀態：管理頁分為「進行中」與「已完成」兩個區域，自動每 2 秒刷新。
    狀態流程：等待中 → 下載中 → 合併中 → 上傳 Drive → 完成 / 失敗
  2. 2 下載中：顯示即時進度條、下載速度（如 5.2 MB/s）、預計剩餘時間（ETA）及已下載大小。
  3. 3 完成後：顯示 Google Drive 連結，點擊 🔗 Drive 即可在瀏覽器開啟並播放影片。
  4. 4 重試失敗：下載失敗的記錄會顯示錯誤原因及 ↻ 重試 按鈕，點擊自動使用原設定重新下載。
  5. 5 刪除記錄：點擊 🗑️ 可刪除下載記錄（不刪除本地檔案及 Google Drive 上的檔案）。
💡 Telegram 也會自動收到通知，無需一直留在管理頁等候。
📋 活動日誌
  1. 1 概覽統計：頁面頂部顯示日誌總數、今日數量、成功 / 錯誤計數。
  2. 2 篩選日誌：可按「類別」篩選：下載 上傳 分析 認證 分類 系統；可按「等級」篩選：INFO / SUCCESS / WARN / ERROR。
  3. 3 查看詳情：點擊任意日誌條目展開詳細資訊（如下載 URL、錯誤訊息、Drive 連結等）。
  4. 4 載入更多：日誌每次顯示 50 條，點擊底部「載入更多」可載入更舊的記錄（最多保存 500 條）。
  5. 5 清除日誌：點擊右上角「清除」按鈕可清空所有日誌記錄（不可恢復）。
⚙️ 系統設定
  1. 1 字體大小：選擇「中」（預設 16px）、「大」（~19px）或「特大」（~22px）。設定即時生效並永久保存，適合不同螢幕尺寸。
  2. 2 YouTube Cookies：如需下載年齡限制或會員內容，需上傳從瀏覽器匯出的 cookies.txt。
    匯出方法：安裝 Chrome 擴充 Get cookies.txt LOCALLY，登入 YouTube 後點擊匯出。
  3. 3 修改帳號：輸入目前密碼 → 填寫新用戶名及/或新密碼（兩次確認）→ 點擊「💾 儲存帳號設定」。所有 Session 將立即失效。
  4. 4 登出：點擊「🚪 登出」按鈕，Session 即時失效並跳轉至登入頁。
  5. 5 Telegram 通知：填寫 Group ID 及（可選）Topic ID → 點「💾 儲存設定」→ 開啟啟用開關。詳見 📲 Telegram 通知設定指南。
❓ 常見問題
  1. Q 分析失敗，提示「Sign in to confirm you're not a bot」
    → 需要上傳 YouTube Cookies。前往 上傳 cookies.txt。
  2. Q 下載後看不到 Google Drive 連結
    → Drive 上傳可能失敗。查看 的「上傳」類別，確認錯誤原因（常見：Drive 容量已滿）。
  3. Q 下載速度慢 / 被限速
    → 屬於 YouTube 伺服器端限速，為正常現象。高畫質影片（4K / 1080p）需要較長下載時間。
  4. Q 選擇的畫質（如 4K）不在列表中
    → 原始影片可能沒有該解析度。yt-dlp 只列出影片實際提供的畫質選項。
  5. Q 網站出現 Cloudflare Error 1033
    → Cloudflare Tunnel 服務中斷，請通知管理員重啟 Tunnel。
  6. Q MP4 / MKV / MOV 有什麼分別？
    → 三者均為封裝格式（container），影音內容完全一樣、檔案大小相同，只是「外殼」不同。MP4 兼容性最佳（Windows / macOS / iOS / Android 均支援），建議優先選用。

🤖 用 OpenClaw 從 GitHub 安裝 — 最簡單，推薦給大多數用戶

OpenClaw 是你的 AI 助手機器人。你只需要跟它說一句話，它就會自動從 GitHub 下載程式碼、安裝所有工具、設定好帳號密碼，然後告訴你網址。完全不需要懂任何技術。

📋 你需要有什麼（先決條件）
  • 一個已經設定好的 OpenClaw 機器人（Telegram 裡的 Bot）
  • 機器人所在的伺服器已啟動（VPS 或本機 Linux 電腦）
步驟 1：叫 OpenClaw 從 GitHub 安裝

打開 Telegram，找到你的 OpenClaw 機器人對話窗口，然後複製貼上以下訊息傳送給它：

請幫我從 GitHub 安裝 YouTube Downloader：
git clone https://github.com/tele02nwt/youtube-downloader.git
安裝完成後告訴我網址和登入密碼

OpenClaw 收到後會自動幫你：

  • 從 GitHub 複製程式碼（git clone）
  • 安裝所有需要的套件（npm install）
  • 建立設定檔、設定登入帳號密碼
  • 啟動伺服器（bash start.sh）
步驟 2：等待自動安裝（約 2-3 分鐘）

你不需要做任何操作。OpenClaw 會自動執行以下指令：

# OpenClaw 自動執行，你不需要輸入
cd youtube-downloader && npm install
cp .env.example .env # 建立設定檔
bash start.sh # 啟動伺服器
💡 喝杯茶等一下就好。你會看到機器人的回覆進度，包括正在安裝哪些工具、設定什麼服務等。
步驟 3：取得登入資訊

安裝完成後，OpenClaw 會回覆你以下資訊：

  • 本機網址：http://伺服器IP:3847（同一網絡內的電腦可以直接存取）
  • 登入帳號和密碼（由 OpenClaw 自動產生）

用這個網址和密碼登入，就可以開始使用了！

⚠️ 如果你想從手機或外部網絡存取（例如在公司存取家裡的伺服器），你需要完成下面的步驟 4 來設定 Cloudflare Tunnel。
步驟 4（重要）：設定 Cloudflare Tunnel 外部存取

如果你想從手機或外部網絡存取 YouTube Downloader，必須設定 Cloudflare Tunnel。這會給你一個 HTTPS 網址（例如 https://yt.yourdomain.com），在任何地方都能打開。

  1. 1 在電腦瀏覽器打開 http://伺服器IP:3847，用 OpenClaw 給你的帳號密碼登入
  2. 2 點擊頂部導航列的 分頁
  3. 3 往下滾動，找到 「☁️ Cloudflare Tunnel」 區塊
  4. 4 點擊 「啟動 Tunnel」 按鈕
  5. 5 等待 10-20 秒讓 Tunnel 連線成功
  6. 6 連線成功後，頁面會顯示一個綠色的網址（例如 https://yt.yourdomain.com）— 這就是你的外部存取網址，記得收藏到書籤！
  7. 7 拿出手機，在瀏覽器打開那個網址，測試一下能不能正常登入
💡 如果你沒有自訂域名，Cloudflare 會自動產生一個隨機網址，像是 https://abc-xyz.trycloudflare.com，同樣可以正常使用。
步驟 5（可選）：連結 Google Drive 自動上傳

連結後，每次下載完成的影片都會自動上傳到你的 Google Drive，按分類整理。

  1. 1 在 分頁，往下找到 「📂 Google Drive」 區塊
  2. 2 點擊 「連結 Google Drive」 或 「Authorize」 按鈕
  3. 3 瀏覽器會打開 Google 登入頁面 — 用你的 Google 帳號登入
  4. 4 點擊 「Allow」（允許） 授權存取
  5. 5 回到 App，應該會看到 「已連結」（Connected） 狀態
  6. 6 完成！現在每次下載影片都會自動上傳到你的 Google Drive
💡 影片會按照你的下載分類自動整理到 Google Drive 對應的資料夾裡。
步驟 6（可選）：啟用 Telegram 下載通知

啟用後，每次影片下載完成都會自動發送通知到你的 Telegram 群組，附上檔名和下載連結。需要伺服器上有安裝 OpenClaw CLI。

  1. 1 在 分頁，往下找到 「📲 Telegram 通知」 區塊
  2. 2 將 「啟用 Telegram 通知」 開關切換為 ON
  3. 3 填入以下資訊：
    • Group ID：你的 Telegram 群組 ID（例如 -1001234567890）。不知道？傳送 我的群組 ID 是什麼？ 給 OpenClaw 機器人。
    • Topic ID：如果你的群組有開啟「主題」功能，填入 Topic ID；沒有的話留空即可。
  4. 4 點擊 「儲存設定」
  5. 5 點擊 「測試通知」 驗證是否正常 — 你的 Telegram 群組應該會收到一條測試訊息
  6. 6 完成！之後每次下載完成都會自動收到 Telegram 通知
🔧 遇到問題？
  1. ? 機器人說「找不到技能」：傳送這段訊息給 OpenClaw：
    幫我安裝 youtube-downloader skill
  2. ? 出現 "Cannot find module" 錯誤：傳送給 OpenClaw：

  4. ? Cloudflare Tunnel 連不上：檢查網絡連線是否正常。到 頁的 Cloudflare 區塊，先點「停止」再點「啟動」重試。
📦 OpenClaw 自動安裝了什麼？

你不需要理解這些，但如果好奇的話：

  • ✅ Node.js — 程式運行環境（讓應用程式能執行）
  • ✅ yt-dlp — 影片下載引擎（負責從 YouTube 抓取影片）
  • ✅ ffmpeg — 影音合併工具（把影片和聲音合在一起）
  • ✅ YouTube Downloader — 這個應用程式本身（從 GitHub 下載）
  • ✅ 自動重啟監控 — 每 10 分鐘健康檢查，崩潰後自動恢復（cron 排程）
🐧 Linux 安裝指南 — 從 GitHub 安裝

適合有 Linux 伺服器或 VPS 的用戶。即使你從未接觸過命令列也沒關係，下面會一步步帶你完成。每一步都有詳細說明，照著做就好。

📋 系統需求
  • 作業系統：Ubuntu 20.04 / 22.04 / 24.04、Debian 11/12、或任何現代 Linux
  • 記憶體：至少 1GB RAM
  • 硬碟空間：至少 5GB 可用空間（存影片需要更多）
  • 網絡：能連接到互聯網
💻 如何打開終端機（Terminal）

終端機是一個文字視窗，你會在裡面輸入指令來安裝和設定程式。以下是打開方法：

  • 如果你的伺服器是遠端 VPS（例如 DigitalOcean、Linode）：
    • Windows 用戶：下載並安裝 PuTTY 軟件 → 打開 PuTTY → 在「Host Name」欄位輸入你伺服器的 IP 地址 → 點擊「Open」→ 輸入用戶名和密碼登入
    • macOS 用戶：打開「應用程式」→「工具程式」→「Terminal.app」→ 輸入 ssh 用戶名@伺服器IP 然後按 Enter
  • 如果你直接坐在 Linux 電腦前面：在桌面空白處右鍵 → 開啟終端機；或按鍵盤上的 Ctrl+Alt+T
💡 看到一個閃爍的游標（通常是 username@hostname:~$）就代表終端機已經準備好了。你可以開始輸入指令了。
步驟 1：安裝 Node.js（程式運行環境）

Node.js 讓你的伺服器能運行 YouTube Downloader 應用程式。在終端機裡逐行複製貼上以下指令，每輸入一行就按 Enter：

# 下載 Node.js 安裝腳本（這行指令會自動設定好安裝來源）
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
# 正式安裝 Node.js
sudo apt-get install -y nodejs
💡 如果系統問你密碼，輸入你的登入密碼（輸入時螢幕不會顯示任何字元，這是正常的），然後按 Enter。

安裝完成後，驗證一下是否成功：

node --version

應該會顯示類似 v20.x.x 的版本號。如果有顯示，代表安裝成功。

步驟 2：安裝 yt-dlp 和 ffmpeg（影片下載工具）

這兩個工具負責從 YouTube 下載影片並合併音頻和視頻。在終端機輸入：

# 安裝 ffmpeg（影音合併工具）和 pip（Python 套件管理器）
sudo apt install -y ffmpeg python3-pip
# 安裝 yt-dlp（YouTube 影片下載工具）
pip3 install yt-dlp

這些工具各自負責什麼：

  • yt-dlp — 從 YouTube 抓取影片的工具
  • ffmpeg — 合併影片和聲音的工具（YouTube 的影片和聲音是分開傳送的，需要用它拼在一起）
⚠️ 如果 pip3 提示 "externally-managed-environment" 錯誤，改用：pip3 install --break-system-packages yt-dlp
步驟 3：安裝 Git（程式碼下載工具）

Git 是用來從 GitHub 下載應用程式碼的工具。大多數 Linux 系統已經預裝了，但以防萬一：

sudo apt install -y git
步驟 4（可選）：安裝 cloudflared（外部存取工具）

如果你想從手機或外部網絡存取你的 YouTube Downloader（而不只是在伺服器本機使用），需要安裝 Cloudflare Tunnel：

# 加入 Cloudflare 官方套件來源
curl -L https://pkg.cloudflare.com/cloudflare-main.gpg | sudo tee /usr/share/keyrings/cloudflare-main.gpg > /dev/null
echo "deb [signed-by=/usr/share/keyrings/cloudflare-main.gpg] https://pkg.cloudflare.com/cloudflared any main" | sudo tee /etc/apt/sources.list.d/cloudflared.list
# 安裝 cloudflared
sudo apt update && sudo apt install cloudflared
💡 如果你只在本機使用（直接在伺服器上開瀏覽器），可以跳過這一步。安裝後在 Web UI 的 頁可以啟動 Tunnel。
步驟 5：從 GitHub 下載 YouTube Downloader

逐行輸入以下指令（每輸入一行就按 Enter）：

# 從 GitHub 下載程式碼到你的電腦
git clone https://github.com/tele02nwt/youtube-downloader.git
# 進入程式資料夾
cd youtube-downloader
# 安裝程式需要的配件
npm install
💡 git clone = 把程式碼從網上複製到你的電腦；npm install = 自動安裝程式需要的所有配件。
步驟 6：設定帳號密碼

先複製一份設定範本檔案：

cp .env.example .env

然後用文字編輯器打開它。nano 是 Linux 內建的簡易編輯器：

nano .env

在檔案裡找到並修改以下幾行（用方向鍵移動游標，直接打字就能修改）：

# 登入帳號（用你的 email 地址）
YT_AUTH_USER=你的email@example.com
# 登入密碼（請設定一個安全的密碼）
YT_AUTH_PASS=你的密碼
# 影片存放位置（改成你想要的路徑）
DOWNLOADS_DIR=/home/你的用戶名/downloads/youtube
💡 nano 編輯器操作方法：用鍵盤方向鍵移動游標 → 直接打字修改內容 → 改完按 Ctrl+X（同時按住 Ctrl 和 X）→ 按 Y 確認儲存 → 按 Enter 完成。
步驟 7：啟動應用程式
bash start.sh
⚠️ VPS 用家注意：如果你用的是雲端 VPS（DigitalOcean、Vultr、Hetzner、AWS 等），需要在控制台的防火牆 / Security Group 開放 TCP port 3847（Inbound）。如不想開放防火牆，可用下方「可選步驟 A」的 Cloudflare Tunnel 代替，不需要直接暴露端口。

啟動成功後，打開你電腦的瀏覽器（Chrome、Firefox 等），在網址列輸入：

http://localhost:3847

如果是遠端 VPS，把 localhost 換成你伺服器的 IP 地址，例如 http://123.45.67.89:3847。

✅ 看到登入畫面就代表安裝成功！用你剛才在 .env 設定的帳號密碼登入即可。
步驟 8：存取你的 YouTube Downloader

根據你的情況，存取方式有所不同：

  • 本機使用：直接在瀏覽器輸入 http://localhost:3847
  • 區域網路存取：在同一個 WiFi 網絡的其他裝置，輸入 http://伺服器內網IP:3847（例如 http://192.168.1.100:3847）
  • 外部存取：需要設定 Cloudflare Tunnel（見下方可選步驟 A）
可選步驟 A：☁️ Cloudflare Tunnel（外部存取）

Cloudflare Tunnel 讓你從任何地方（手機、辦公室電腦）存取你的 YouTube Downloader，不需要設定防火牆或端口轉發。

如果你在步驟 4 已安裝 cloudflared，只需要：

  1. 1登入 YouTube Downloader Web UI
  2. 2前往 頁面，找到「Cloudflare Tunnel」面板
  3. 3選擇「⚡ Quick Tunnel」模式（免帳號、最簡單）→ 點擊「🚀 啟動 Tunnel」
  4. 4啟動後會產生一個 *.trycloudflare.com 的公開網址，用任何裝置都能存取
⚠️ Quick Tunnel 的網址每次重啟都會改變。如需固定網址，請參考下方 Token Tunnel 設定。
固定網址 Token Tunnel（適合長期使用）
  1. 1前往 https://one.dash.cloudflare.com 登入（需要免費 Cloudflare 帳號，沒有的話先免費註冊）
  2. 2左側選單 → Networks → Tunnels → 點擊「Create a Tunnel」按鈕
  3. 3選「Cloudflared」→ 輸入 Tunnel 名稱（例如 yt-downloader）→ 點「Save Tunnel」
  4. 4在「Install connector」頁面，複製顯示的 Token（以 eyJ 開頭的長字串）
  5. 5回到 YouTube Downloader Web UI → → Cloudflare Tunnel 面板 → 選「🔑 Token Tunnel」→ 貼上 Token → 點「🚀 啟動 Tunnel」
  6. 6回到 Cloudflare Dashboard → Tunnels → 你的 Tunnel → 點「Configure」→「Public Hostname」→「Add a public hostname」→ 填入網域/子網域，Service 選 HTTP，URL 填 localhost:3847 → 儲存
💡 Token Tunnel 設定後網址永遠固定，伺服器重啟後自動重連，適合長期使用。
可選步驟 B：📂 Google Drive 自動上傳

讓下載完成的影片自動上傳到你的 Google Drive。需要安裝 gog CLI（Google 工具命令列）：

# 下載並安裝 gog CLI（自動偵測 CPU 架構）
ARCH=$(uname -m)
if [ "$ARCH" = "x86_64" ]; then ARCH_TAG="amd64"; else ARCH_TAG="arm64"; fi
VERSION="0.12.0"
curl -L "https://github.com/steipete/gogcli/releases/download/v${VERSION}/gogcli_${VERSION}_linux_${ARCH_TAG}.tar.gz" -o /tmp/gog.tar.gz
tar -xzf /tmp/gog.tar.gz -C /tmp
sudo mv /tmp/gog /usr/local/bin/gog
gog --version

安裝後，在 Web UI 的 頁面：

  1. 1找到「Google Drive」面板，點擊「🔗 開始授權」
  2. 2面板中會出現一個授權網址 — 用任何瀏覽器打開（手機也可以）完成 Google 登入
  3. 3授權完成後回到設定頁，狀態會顯示「已連接 Google Drive ✓」
💡 之後每次下載完成，影片會自動上傳到你的 Google Drive，並在管理頁顯示 Drive 連結。
可選步驟 C：📲 Telegram 通知

讓系統在每次下載完成後，自動發送 Telegram 通知。此功能需要伺服器上已安裝 OpenClaw CLI。

在 Web UI 的 頁面：

  1. 1找到「Telegram 通知」面板
  2. 2填入你的 Telegram Group ID（以 -100 開頭的數字）和 Topic ID（如果是 Forum 群組）
  3. 3點擊「📨 發送測試訊息」確認通知正常
  4. 4開啟自動通知開關即可
⚠️ 如果沒有安裝 OpenClaw CLI，此功能將無法使用。聯絡管理員安裝 OpenClaw。
建議：設定開機自動啟動

讓伺服器重啟後自動啟動 YouTube Downloader，不用每次手動執行：

# 打開排程設定
crontab -e
# 在最後一行加入：
@reboot bash ~/youtube-downloader/start.sh

~ 代表你的 home 目錄，免去填寫用戶名。如果你把程式安裝在其他位置，請改成完整路徑。

💡 如果 crontab -e 問你要用什麼編輯器，選 nano（通常是選項 1）。
🪟 Windows 安裝指南 — 從 GitHub 安裝

在 Windows 電腦上從零開始安裝 YouTube Downloader。提供兩種方式，強烈建議使用方式 A（WSL2），因為它支援所有功能。

📋 Windows 版本要求

需要 Windows 10 版本 2004 或以上（Windows 11 也可以）。

怎麼檢查你的版本？同時按下鍵盤上的 Win+R（Win 是左下角那個方塊圖案鍵），在彈出的小視窗裡輸入 winver，按 Enter，就能看到你的 Windows 版本號。

✅ 方式 A：WSL2 + Ubuntu（強烈推薦，支援所有功能）

WSL2 是什麼？你可以把它想像成在你的 Windows 電腦裡放了一個 Linux 小房間。YouTube Downloader 在這個小房間裡運行，但你可以直接用 Windows 的瀏覽器存取它。這個方式支援所有功能，包括 Google Drive 上傳和 Telegram 通知。

  1. 1 安裝 WSL2：
    • 按 Windows 鍵（鍵盤左下角那個方塊圖案），輸入「PowerShell」搜尋
    • 在搜尋結果中，右鍵點擊「Windows PowerShell」→ 選擇「以系統管理員身分執行」
    • 看到藍色視窗後，輸入以下指令，然後按 Enter：
      wsl --install
    • 等待安裝完成（可能需要幾分鐘），然後重啟電腦
  2. 2 設定 Ubuntu 用戶：重啟電腦後，一個黑色的 Ubuntu 視窗會自動彈出。等待安裝完成後，它會要求你：
    • 輸入一個用戶名（可以用你的英文名，全小寫，例如 david）
    • 設定一個密碼（這是 Ubuntu 裡的密碼，跟 Windows 帳號無關）
    ⚠️ 密碼輸入時螢幕不會顯示任何字元，這是正常的安全措施。盲打輸入你的密碼後按 Enter 即可。
    💡 如果重啟後 Ubuntu 視窗沒有自動彈出，按 Windows 鍵 → 在搜尋欄輸入「Ubuntu」→ 點擊「Ubuntu」圖示打開。
  3. 3 安裝 Node.js：在 Ubuntu 視窗裡逐行輸入：
    curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
    sudo apt-get install -y nodejs
    💡 如果問你密碼，輸入剛才在步驟 2 設定的 Ubuntu 密碼。
  4. 4 安裝 yt-dlp 和 ffmpeg：
    sudo apt install -y ffmpeg python3-pip git
    pip3 install yt-dlp
    ⚠️ 如果 pip3 提示 "externally-managed-environment" 錯誤，改用：pip3 install --break-system-packages yt-dlp
  5. 5 安裝 cloudflared（可選，用於外部存取）：
    curl -L https://pkg.cloudflare.com/cloudflare-main.gpg | sudo tee /usr/share/keyrings/cloudflare-main.gpg > /dev/null
    echo "deb [signed-by=/usr/share/keyrings/cloudflare-main.gpg] https://pkg.cloudflare.com/cloudflared any main" | sudo tee /etc/apt/sources.list.d/cloudflared.list
    sudo apt update && sudo apt install cloudflared
    💡 如果你只在自己電腦上使用，可以跳過這一步。
  6. 6 從 GitHub 下載應用程式：逐行輸入：
    git clone https://github.com/tele02nwt/youtube-downloader.git
    cd youtube-downloader
    npm install
  7. 7 設定帳號密碼：
    cp .env.example .env
    nano .env
    在編輯器裡修改 YT_AUTH_USER（你的 email）和 YT_AUTH_PASS（你的密碼）。
    nano 操作：方向鍵移動游標 → 直接打字修改 → Ctrl+X → Y → Enter 儲存退出。
  8. 8 啟動應用程式：
    bash start.sh
  9. 9 打開 Windows 瀏覽器：在 Windows 的 Chrome / Edge / Firefox 中，網址列輸入：
    http://localhost:3847
    WSL2 會自動把端口共享給 Windows，直接就能存取，不需要額外設定。
  10. 10 可選步驟：安裝 gog CLI（Google Drive 上傳功能）

    在 Ubuntu 視窗裡逐行輸入：

    ARCH=$(uname -m)
    if [ "$ARCH" = "x86_64" ]; then ARCH_TAG="amd64"; else ARCH_TAG="arm64"; fi
    VERSION="0.12.0"
    curl -L "https://github.com/steipete/gogcli/releases/download/v${VERSION}/gogcli_${VERSION}_linux_${ARCH_TAG}.tar.gz" -o /tmp/gog.tar.gz
    tar -xzf /tmp/gog.tar.gz -C /tmp
    sudo mv /tmp/gog /usr/local/bin/gog
    gog --version
    安裝後，打開 YouTube Downloader Web UI → ⚙️ 設定 → Google Drive → 點擊「🔗 開始授權」→ 會顯示一個 URL，用 Windows 瀏覽器開啟該 URL 完成 Google 帳號授權即可。
✅ 安裝完成！之後每次要啟動，只要打開 Ubuntu 視窗，輸入 cd youtube-downloader && bash start.sh 即可。

# 路徑視乎你的安裝位置而定
YT_DLP_PATH=C:\Users\你的用戶名\AppData\Local\Microsoft\WinGet\Links\yt-dlp.exe
FFMPEG_PATH=C:\Users\你的用戶名\AppData\Local\Microsoft\WinGet\Links\ffmpeg.exe
💡 如果路徑不對，可以打開 PowerShell 輸入 where.exe ffmpeg 和 where.exe yt-dlp 查看實際路徑，再填入 .env 檔案。
⚙️ 安裝後可選設定

安裝完成並登入後，可以在 頁面配置以下功能：

  • ☁️ Cloudflare Tunnel — 從手機或外部網絡存取（Quick Tunnel 免帳號最簡單）
  • 📂 Google Drive — 自動上傳影片（僅 WSL2 方式支援）
  • 📲 Telegram 通知 — 下載完成後自動通知（需要 OpenClaw CLI，僅 WSL2 方式支援）
💡 建議：設定開機自動啟動（方式 A WSL2）

讓電腦重啟後自動啟動 YouTube Downloader，不用每次手動執行。

在 Ubuntu 視窗裡輸入：

crontab -e

如果問你用什麼編輯器，選 nano（輸入對應數字按 Enter）。

在最後一行加入：

@reboot bash ~/youtube-downloader/start.sh

然後 Ctrl+X → Y → Enter 儲存。

💡 ~ 代表你的 home 目錄，不需要改用戶名。之後每次重啟電腦後，只要打開 Ubuntu 視窗（不需要輸入任何指令），伺服器就會自動在背景啟動。
